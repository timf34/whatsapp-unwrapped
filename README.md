# WhatsApp Unwrapped

[https://www.granola.ai/updates/granola-crunched](https://www.granola.ai/updates/granola-crunched) but for Whatsapp convo's. 98% there, just need to make it funnier - come back again soon. 

## Setup

1. Create and activate the virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
